# 🌻 Flores Amarillas (Yellow Flowers) - Mobile Page

## 🇪🇸 Español

### Descripción General
Una página web móvil diseñada como un regalo interactivo y personalizado. Este proyecto combina un diseño visual atractivo con elementos dinámicos, destacando un fondo aleatorio interactivo y un contador de tiempo en vivo. 

### Características Principales
- **Fondo Dinámico:** Generación aleatoria de emojis en el fondo para crear una experiencia visual única en cada visita.
- **Contador de Tiempo:** Un algoritmo que calcula en tiempo real los días, meses y años transcurridos desde una fecha especial (11 de julio de 2025) hasta la actualidad.
- **Mobile-First:** Diseño responsivo enfocado exclusivamente en ofrecer una experiencia fluida, rápida y ligera en dispositivos móviles.

### Tecnologías Utilizadas
- **HTML5 & CSS3:** Estructura, maquetación y diseño visual.
- **JavaScript (Vanilla):** Lógica del contador de fechas y manipulación del DOM para la generación aleatoria del fondo.

### Propósito del Proyecto
Este repositorio fue creado como un proyecto personal (un regalo para mi novia, Camila) y sirve como demostración práctica de mis habilidades en el desarrollo Front-end sin depender de librerías externas. 

> **Nota:** Este proyecto está finalizado y se mantiene público con fines de portafolio y exhibición de código. Actualmente no se aceptan *Pull Requests*.

## 🇬🇧 English

### Overview
A mobile web page designed as a personalized, interactive gift. This project combines an appealing visual design with dynamic elements, featuring an interactive randomized background and a live time counter.

### Key Features
- **Dynamic Background:** Randomized emoji generation in the background to create a unique visual experience on every visit.
- **Live Time Counter:** A real-time algorithm that calculates the days, months, and years passed since a special date (July 11, 2025) to the present day.
- **Mobile-First:** Responsive design focused exclusively on delivering a smooth, fast, and lightweight experience on mobile devices.

### Technologies Used
- **HTML5 & CSS3:** Structure, layout, and visual design.
- **JavaScript (Vanilla):** Date counter logic and DOM manipulation for the random background generation.

### Project Purpose
This repository was created as a personal project (a gift for my girlfriend, Camila) and serves as a practical showcase of my Front-end development skills without relying on external libraries.
